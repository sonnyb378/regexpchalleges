Problem 1: Extract all words (sequences of alphabetic characters) from a given text.
Input: "Hello World! This is a simple text."
Output: ["Hello", "World", "This", "is", "a", "simple", “text"]

Problem 2: Check if a string contains only digits (0-9).
Input (Correct): "12345"
Input (Incorrect): "12A34"
Output (Correct): true
Output (Incorrect): false

Problem 3: Extract all email addresses from a given text.
Input: "Please contact support@example.com for assistance or info@domain.co.uk."

Problem 4: Check if a string is a valid URL (starting with "http://" or "https://").
Input (Correct): "https://www.example.com"
Input (Incorrect): "ftp://example.com"

Output (Incorrect): false

Problem 5: Extract all words that are in uppercase from a given text.
